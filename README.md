# Filum

Social-first platform bridging Threads/Twitter and Reddit — upvote/downvote on posts and comments, dual feeds, threaded conversations.

**Stack:** SvelteKit · Tailwind CSS · Supabase · Vercel

## Dev

```sh
bun install
bun run dev
```

## Deploy

Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in Vercel env vars, then push to `main`.
